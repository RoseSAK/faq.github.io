Manager Guide

This is the itracr Manager guide. The Manager feature allows administrators to assign managers to users, along with certain permissions. Manager permissions include the ability to approve annual leave, sick leave and timecards.

Assigning a Manager

This section explains how an administrator can assign a manager to a user.

Navigate to the information page for a user, via the Users page. Click on the Edit button. Below the User Permissions section, you will see a dropdown menu labelled Manager. You can then select a manager for this user from a list of existing users.

Manager Permissions

Once you have selected a manager, three checkboxes will appear. Selecting each of these checkboxes will give the manager permission to approve this user's timecard, annual leave and sick leave respectively. Select the correct permissions, and then click 'Save User'.

Manager Approvals

A manager who has been assigned one or more approval permissions for a user/multiple users will act as the first point of approval for that user. For example, when the user submits a timecard, the manager will be able to view and approve the timecard, and also add notes. The timecard will still need to be approved by an administrator, but the administrator will be able to see that the manager has approved/queried/rejected the timecard, and also the reason why.

Management Menu

On the itracr homepage, a manager will have an extra dropdown menu called Management.
